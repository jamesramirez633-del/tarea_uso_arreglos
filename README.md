# tarea_uso_arreglos  
#descripcion
En este programa vamos aver como se utilizan los arreglos calculando las calificaciones de estudiantes, sacando el promedio y mostrando la mas alta y la mas baja.


#Ejemplo:

Cantidad de estudiantes: 2

Estudiante 1:
80 70 90 100

Estudiante 2:
60 65 50 55

Promedio general: 71.25
Aprobados: 1
Reprobados: 1

# uso de arreglos
 Un arreglo es una estructura que guarda muchos datos del mismo tipo en una sola variable. En este codigo se utilizan arreglos bidimensionales para almacenar las notas de cada estudiante y arreglos unidimensionales para calcular promedios, el programa puede recorrer fácilmente todos los datos para ingresarlos, procesarlos y mostrarlos, haciendo el trabajo más rápido y ordenado sin necesidad de usar muchas variables individuales.
